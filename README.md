This is an archived export of the Rust issue tracker up to 2016.

It contains three types of files:

- `000000-main.json`.
  The original issue/PR post.
  Raw dump of the `/repos/rust-lang/issues/{num}` API endpoint.
- `000000-comments.json`.
  Subsequent comments.
  Raw dump of the `/repos/rust-lang/issues/{num}/comments` API endpoint.
- `000000-xrefs.json`.
  Cross-references between issues.
  Filtered from the very large `/repos/rust-lang/issues/{num}/timeline` API endpoint.

Fetched on 2025-12-30.
